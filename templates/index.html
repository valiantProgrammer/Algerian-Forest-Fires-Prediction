Mathematical Framework & Equations

This section presents the mathematical foundations of the regression models used in this analysis. Each equation is essential to understanding how the models learn patterns in the data.

Linear Regression Model

The foundation of all regression models - assumes a linear relationship between features and target variable.

$$\hat{y} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n$$
Mean Squared Error (MSE)

Cost function that measures the average squared difference between predicted and actual values.

$$J(\beta) = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2$$
Ridge Regression (L2 Regularization)

Adds L2 penalty to prevent overfitting by constraining coefficient magnitude. Best performing model for this dataset.

$$J(\beta) = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2 + \lambda \sum_{j=1}^{n} \beta_j^2$$
Lasso Regression (L1 Regularization)

Uses L1 penalty for feature selection by driving some coefficients to exactly zero.

$$J(\beta) = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2 + \lambda \sum_{j=1}^{n} |\beta_j|$$
Elastic Net (Combined Regularization)

Combines Ridge and Lasso penalties for balanced regularization between L1 and L2.

$$J(\beta) = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2 + \lambda \left[ \alpha \sum_{j=1}^{n} |\beta_j| + (1-\alpha) \sum_{j=1}^{n} \beta_j^2 \right]$$
Mean Absolute Error (MAE)

Measures average absolute difference between predictions and actual values. Robust to outliers.

$$MAE = \frac{1}{m} \sum_{i=1}^{m} |y^{(i)} - \hat{y}^{(i)}|$$
Root Mean Squared Error (RMSE)

Square root of MSE. Penalizes larger errors more heavily. Measured in same units as target.

$$RMSE = \sqrt{\frac{1}{m} \sum_{i=1}^{m} (y^{(i)} - \hat{y}^{(i)})^2}$$
R¬≤ Score (Coefficient of Determination)

Measures proportion of variance in target explained by model. Ranges from 0 to 1, where 1 is perfect prediction.

$$R^2 = 1 - \frac{\sum_{i=1}^{m} (y^{(i)} - \hat{y}^{(i)})^2}{\sum_{i=1}^{m} (y^{(i)} - \bar{y})^2}$$
Standard Scaler Normalization

Transforms features to have mean=0 and standard deviation=1. Essential for regularized models.

$$x_{scaled} = \frac{x - \mu}{\sigma}$$
Cross-Validation Error

Average error across k-folds for robust model evaluation and hyperparameter selection.

$$CV_{error} = \frac{1}{k} \sum_{i=1}^{k} Error_i$$

Key Mathematical Concepts

Regularization Parameter (Œª): Controls the strength of the penalty term. Higher Œª results in simpler models but potentially higher bias. Lower Œª allows more complex models with potential overfitting.

Gradient Descent: Iterative optimization algorithm that adjusts coefficients (Œ≤) to minimize the cost function by moving in the direction of steepest descent.

Bias-Variance Tradeoff: Ridge and Lasso add bias to reduce variance, preventing overfitting. Our models achieve excellent R¬≤ (0.98+) through this tradeoff.

Feature Scaling: Standardization ensures all features contribute equally to the model, preventing high-magnitude features from dominating the learning process.
